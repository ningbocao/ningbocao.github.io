搭建属于自己的的博客

发表于 2016-07-02   |   分类于 生活   |  

初衷

为什么要费力的搭建自己的博客?

  1. 工科男爱瞎折腾的毛病；
  2. 讨厌博客平台花里胡哨的主题；
  3. 能有一个自己独立的空间，系统的归纳纪录知识；
  4. 好吧我承认我就是想装装B.

纸上来的终觉浅，绝知此事要躬行

好了动机有了，下面说说怎么搭建博客
首先讲一下我的是mac ios系统。怎么搭建博客的细节我就不在赘述了，已经有很多前辈有过详细的教程和搭建步骤。反正我的博客是写给自己看的，所以本着总结的出发点，我大致罗列的主要的步骤和我遇到的问题和自己的一点感想

购买域名

首先你要购买一个域名，推荐狗爹（godaddy）有各种类型的域名可以选择，根据个人需要了，我购买的是三年的.com大概200RMB,因为当时是用欧元结算的，大概也就这个数。

注册Github

为什么要注册Github呢？因为你需要在网络上有一个空间存储你的博客资料，而Github能够为你免费提供300Mb的空间就博客而言目前够用了。

下载安装HEXO

HEXO是一款基于Node.js的静态博客框架。

初始化设置

在完成上面三步后，你会得到在你所在目录下的博客文件：

├── _config.yml
├── db.json
├── node_modules
├── package.json
├── public
├── scaffolds
├── source
└── themes

常见命令

sudo npm install hexo-deployer-git –save
加sudo是是以管理员身份运行命令

hexo g
生成部署文件

hexo s
生成部署文件

hexo d
部署到Github上

结束语

Je

##问题

###无法识别hexo
在npm install hexo-deployer-git –save命令前加sudo

###无法部署到github
1.检查ssh key
2.检查配置文件

ssh -T git@github.com
caoningbo$ git config –global user.name”ningbocao”
caoningbo$ git config –global user.email “ningbo.cao@buaa.edu.cn”

deploy:
type: git
repository: http://github.com/ningbocao/ningbocao.github.io.git
branch: master

##如何选择新的主题
首先
$ hexo clean

$ hexo g
$ hexo s
进入themes文件夹
然后复制主题
git clone https://github.com/cnfeat/cnfeat.git themes/jacman

修改Hexo目录下的config.yml配置文件中的theme属性，将其设置为jacman
theme: jacman

然后进入主题文件夹，发布主题
$ cd themes/jacman
$ git pull

最后hexo g
hexo s
hexo d
